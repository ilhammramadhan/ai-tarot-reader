# Mystic Tarot

An AI-powered tarot card reading web application with mystical dark theme and 3-card spreads (Past/Present/Future). Built with React, TypeScript, and Google Gemini AI.

![TypeScript](https://img.shields.io/badge/TypeScript-5.9-3178C6?style=flat-square&logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.0-38B2AC?style=flat-square&logo=tailwind-css)
![Gemini AI](https://img.shields.io/badge/Gemini-AI-4285F4?style=flat-square&logo=google)

## Features

- **3-Card Spread Reading** - Past, Present, and Future layout
- **AI-Powered Interpretations** - Mystical readings generated by Google Gemini
- **Card Flip Animations** - Dramatic reveal effects with smooth transitions
- **Mystical Dark Theme** - Deep purple and gold color scheme
- **Light/Dark Mode Toggle** - Switch between mystical themes
- **22 Major Arcana Cards** - Complete major arcana deck
- **Responsive Design** - Works on all devices
- **Copy Reading** - Share your reading with others

## Demo

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Google Gemini API key (free tier available)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/mystic-tarot.git
cd mystic-tarot
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory:
```bash
cp .env.example .env
```

4. Add your Gemini API key to `.env`:
```
VITE_GEMINI_API_KEY=your_api_key_here
```

5. Get your free API key from [Google AI Studio](https://aistudio.google.com/app/apikey)

6. Start the development server:
```bash
npm run dev
```

7. Open [http://localhost:5173](http://localhost:5173) in your browser.

## How It Works

1. **User Input**: Enter your question for the cards
2. **Card Drawing**: 3 random cards are drawn from the Major Arcana
3. **Animation**: Cards flip one by one with staggered timing
4. **AI Reading**: Gemini generates a mystical interpretation based on:
   - Your specific question
   - The 3 cards drawn
   - Their positions (Past/Present/Future)
5. **Display**: Reading appears with copy and new reading options
